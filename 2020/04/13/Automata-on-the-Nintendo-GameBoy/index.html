Apr 13, 2020 development

It's been a while.

I recently saw the sad news that John Conway has passed away. This reminded me of a time in 2012 when I wrote a Cellular Automata "game" for the Nintendo GameBoy as part of the Ludum Dare game jam.

I had to make a few improvements tonight before uploading it, and I am very impressed with the fact that I can spin up a Docker container and install the Gameboy Dev KIT rpm file I have had since then and it allowed me to compile a new version of the rom without any issues. What a time to be alive.

Here you can download it and have a play.

RIP Mr Conway.
