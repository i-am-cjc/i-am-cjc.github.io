cjc.im / projects / Site Security Toolkit

Browser addon that will add an entry to the context menu to scan a site with various online services.

Currently implemented sites:

  • Mozilla Observatory
  • SSL Labs
  • SecurityHeaders.io
  • HSTS Preload
  • CryptCheck
  • Sucuri Sitecheck
  • Geoip Lookup
  • Shodan.io
  • Censys.io

Please note the current shodan request will send the site to scan to a thirdparty to resolve the IP address.

Available to install at:

  • https://addons.mozilla.org/en-US/firefox/addon/site-security-tooklit/
